## RANGY SOCIAL APP

`Backend API for RANGY social app using Nest Framework (NodeJS + TypeScript + PostgreSQL)`

![NodeJS workflow](https://github.com/vikassharma96/rangy-social-backend/actions/workflows/node.yml/badge.svg)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
